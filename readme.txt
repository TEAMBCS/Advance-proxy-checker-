# Advance Proxy Checker

**Advance Proxy Checker** is a fast and powerful tool developed by **BCS** for checking the validity of proxy servers.  
It supports multithreaded proxy validation to improve performance and filter out invalid proxies efficiently.

---

## Features:

- Checks HTTP/HTTPS proxies
- Identifies valid and invalid proxies
- Supports multithreading for faster checking
- Saves working proxies to a separate output file

---

## Usage:

Run the following command in your terminal:

### Arguments:

- `-i proxy.txt` : Input file containing a list of proxies (one per line)
- `-o valid_proxy.txt` : Output file where valid proxies will be saved
- `-t 50` : Number of threads to use for checking (e.g., 50)

---

## Note:

Make sure the `proxy.txt` file contains proxies in the correct format, like:
192.168.0.1:8080
123.45.67.89:3128
